Checkbox

Hinweise

Allgemein

Eine Checkbox (Kontrollkästchen) ist eine Komponente, mit der Nutzer:innen den Wert auswählen können, der in dem entsprechenden Label (Textbeschriftung) steht. Eine Checkbox kann einzeln, z.B. um AGBs zu akzeptieren, und in einer Gruppe verwendet werden. Dann können Nutzer:innen mehrere Kontrollkästchen aus einer Gruppe gleichzeitig auswählen.

Verwendung

Diese Komponente sollte nur dann verwendet werden, wenn es eine kleine bis mittlere Anzahl von Optionen gibt und Nutzer:innen eine oder mehrere Optionen auswählen sollen. Wenn sehr viele Optionen möglich sind, empfiehlt es sich die Dropdown Komponente anzuwenden. Gibt es bei mehreren Optionen nur eine Auswahlmöglichkeit bieten sich folgende Komponenten als Alternative an

  • Radio Button
  • Toggle Button
  • Switch

Entwicklung

Default werden die Checkboxen untereinander platziert. Über die Klasse is-inline, können sie aber auch nebeneinander platziert werden.
