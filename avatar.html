Figma

Avatare werden verwendet, um eine Person oder ein Objekt darzustellen. Folgende Variationen können verwendet werden:

  • Bild
  • Initialen
  • Icon

Folgende Klassen können auf einen Avatar gesetzt werden:

  • c-avatar--auto - Setzt die Höhe und Breite auf maximal 150px

Folgende Klassen können auf einen Avatar-Group gesetzt werden:

  • c-avatar-group--overlay - Überlappt die Avatare
