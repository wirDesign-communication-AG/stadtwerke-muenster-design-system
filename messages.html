Messages

Messages liefern dem Benutzer unmittelbares Feedback. Diese Informationen können Fehler, Bestätigungen, Warnungen oder Hinweise sein.

Folgende Klassen können auf eine Message gesetzt werden:

  • is-success - Success
  • is-info - Info
  • is-warning - Warning
  • is-error - Error
Bitte akzeptieren Sie die Marketing-Cookies, um diesen Inhalt anzuzeigen.
