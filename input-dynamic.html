Input

Input Elemente werden verwendet, um Benutzereingaben zu sammeln. Die Benutzereingaben werden meist zur Verarbeitung an einen Server gesendet.

Bitte akzeptieren Sie die Marketing-Cookies, um diesen Inhalt anzuzeigen.

Text
